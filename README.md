# my1st-etextbook-boilerplate
My1st eTextbook Boilerplate in 2025 by Harry Longworth - Ebook Textbook Template, Boilerplate and Example Document Structuring using Markdown, VSCode, Pandoc and Github for Kindle Self Publishing

The actual guide can be found in the repo here:

[my1st eTextBook boilerplate](./my1st-etextbook-boilerplate.md)

The master copy of this guide is a Markdown file.

Test conversions can be found in the dist folder.

[Dist for Distribution](dist/)

These conversions have been created using Pandoc:  <https://pandoc.org/>

Example conversions:

- MS Word .docx.
- ePub.
- Kindle .kcb.
- HTML.
- more soon.
